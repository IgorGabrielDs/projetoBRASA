FDS (Software Development Fundamentals) project — 2nd period, CESAR School

Stack: Python · Django

Description

Smart News Assistant — your co-pilot for discovering, understanding, and organizing what matters.

Log in to activate filters by topic, region, and sources

Receive personalized recommendations that learn from your usage and ratings (👍/👎)

View dashboards like “Top 3 Most Viewed Stories of the Week”

Save materials to “Read Later” so you don’t miss a thing

With one tap, the “Summarize” button transforms long texts into clear bullet points, maintaining links and context

In each conversation, the chatbot centralizes your news consumption: filtering, recommending, ranking, summarizing, and organizing — all so you spend less time searching and more time understanding.
